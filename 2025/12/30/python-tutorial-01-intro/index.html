本系列教程将带你从零开始学习Python编程，无需任何编程基础。

学习目标

完成本章后，你将学会：

  • 什么是Python以及为什么要学习它
  • 如何在你的电脑上安装Python
  • 如何运行Python代码（交互式模式和脚本模式）
  • 编写你的第一个Python程序

什么是Python？

Python是一种高级编程语言，由荷兰程序员Guido van Rossum在1991年创建。它的名字来源于英国喜剧团体”Monty Python”（蒙提·派森），而不是蟒蛇。

为什么选择Python？

想象一下，如果你要用一种语言和世界各地的朋友交流，你会选择哪种语言？你会选择一种语法简单、表达清晰、容易学习的语言，对吧？

Python就是编程世界的这种语言：

  • 简单易学：Python的语法接近英语，即使没有编程基础也能快速理解
  • 功能强大：从网站开发到人工智能，从数据分析到自动化办公，Python无所不能
  • 社区活跃：全球有数百万Python开发者，遇到问题很容易找到解决方案
  • 跨平台：无论是Windows、Mac还是Linux，Python都能运行

Python的应用领域

领域 应用案例
网站开发 YouTube、Instagram、豆瓣都使用Python
数据科学 数据分析、机器学习、人工智能
自动化 自动化办公、爬虫、测试脚本
游戏开发 游戏逻辑、服务器端

安装Python

在开始编写Python代码之前，我们需要先在电脑上安装Python。

检查是否已安装Python

你的电脑可能已经安装了Python。让我们先检查一下：

Windows用户：

  1. 按 Win + R 键，输入 cmd，按回车
  2. 输入 python --version 或 python3 --version
  3. 如果看到版本号（如 Python 3.8.x），说明已安装

Mac用户：

  1. 打开”终端”（在”应用程序” → “实用工具”中）
  2. 输入 python3 --version
  3. 如果看到版本号，说明已安装

Linux用户：
大多数Linux系统默认已安装Python 3。在终端输入 python3 --version 检查。

如果看到类似 Python 3.8.2 的版本号，恭喜你！你可以跳过安装步骤。

安装Python（如果需要）

Windows安装步骤：

  1. 访问 Python官网
  2. 点击 “Download Python” 按钮（会自动检测你的系统）
  3. 重要：安装时勾选 “Add Python to PATH”
  4. 点击 “Install Now”

警告：如果没有勾选 “Add Python to PATH”，后续需要手动配置环境变量，会比较麻烦。

常见错误

错误1：忘记勾选 “Add Python to PATH”

症状：在命令提示符输入 python 或 python3 时提示”不是内部或外部命令”。

解决方案：

  1. 重新运行Python安装程序
  2. 这次务必勾选 “Add Python to PATH”
  3. 或手动将Python添加到系统PATH

正确做法：

  • Python使用缩进表示代码块（类似于段落）
